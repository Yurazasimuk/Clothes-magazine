# Fashion Store

## Опис проєкту
Fashion Store – це веб-застосунок для перегляду та покупки товарів онлайн. Користувачі можуть реєструватися, додавати товари у кошик, переглядати каталог товарів як у вигляді карток, так і текстовим списком. Адміністратор має можливість додавати нові товари та редагувати існуючі.  

## Мета
Метою проєкту є створення простого і стильного веб-магазину з базовим функціоналом для користувачів та адміністратора, який демонструє основи роботи з локальним збереженням даних і динамічним відображенням контенту.  

## Використані технології
- **HTML** – структура сторінок  
- **CSS** – оформлення та стиль інтерфейсу  
- **JavaScript** – логіка роботи додатку, кошик, адміністрування  
- **localStorage** – зберігання даних про товари та кошик  

## Інструкція запуску
1. Клонуйте або завантажте репозиторій на свій комп’ютер.  
2. Відкрийте файл `index.html` у будь-якому сучасному браузері.  
3. Для перегляду кошика відкрийте `cart.html`, для адмінки – `admin.html`, для реєстрації та входу – `register.html` та `login.html`.
